Fall 2023 · CS 188 · Artificial Intelligence

AI Pacman Solver

Berkeley CS 188's classic Pacman project series — implementing AI agents progressively from search to adversarial planning to learning, all running against the same Pacman game environment.

Overview

CS 188 (Introduction to Artificial Intelligence) uses Pacman as the testing ground for every major AI technique. The same game gets attacked from multiple angles — uninformed search, informed search with heuristics, adversarial planning against ghosts, probabilistic inference under uncertainty, and finally reinforcement learning agents that learn from game experience.

What I Built

  • Search algorithms: BFS, DFS, UCS, and A* with admissible heuristics for solving maze pathfinding variants.
  • Adversarial agents: minimax and α-β pruning for multi-agent Pacman-vs-ghost games with bounded lookahead.
  • Probabilistic inference: Bayes-net tracking of ghost positions from noisy distance sensors.
  • Reinforcement learning: Q-learning and approximate Q-learning agents trained on Pacman gameplay.
