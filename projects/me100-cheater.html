"The Cheater" — ESP32 IoT Project

My first embedded IoT project, built in ME 100 — the entry point into the ESP32 + MicroPython hardware stack I've used on every project since.

Overview

ME 100 (Electronics for IoT) is Berkeley's hands-on intro to embedded firmware and wireless-connected hardware. "The Cheater" was my course project — and where my hardware journey actually started. Several years and many builds later, the ESP32 + MicroPython stack I picked up here still underpins my current robotics and sensor work.

What I Built

  • ESP32-based IoT device built as part of the ME 100 course project.
  • MicroPython firmware on Espressif hardware.
  • Wireless connectivity for remote command and control.
  • Hands-on integration of circuits, sensors, and embedded firmware — the toolchain I've extended in every project since.
