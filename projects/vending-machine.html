Fall 2023 · ME 102B · Mechatronics

Vending Machine

Built a fully functional vending machine as the ME 102B Mechatronics capstone — integrating mechanical design, embedded firmware, sensor input, motor control, and user interface into one working product.

Overview

ME 102B's capstone forces a small team to ship a working mechatronic product from scratch. We built a vending machine — a deceptively complex problem that touches every discipline the course teaches: mechanism design, motor control, sensor selection, embedded state machines, and user-facing interaction.

What I Built

  • Mechanical: dispensing mechanism, product loading system, structural enclosure (designed in CAD, fabricated and assembled).
  • Firmware: microcontroller code driving motors, polling sensors, and managing the vending state machine.
  • Sensor integration: product detection and feedback signals for closing the dispense loop.
  • User interface: input handling and visual feedback for the end user.
