V-JEPA Object Tracking

Latent-space prediction of projectile trajectories through visual occlusions using V-JEPA 2.1 with CNN decoders — tracking objects the camera can't actually see.

Overview

A computer vision project exploring whether self-supervised video models can reason about object trajectories that pass through occlusions — situations where the camera briefly loses sight of the target but physics says it must still be moving.

The system uses V-JEPA 2.1 as a frozen latent predictor and trains a small CNN decoder to extract the projectile's spatial position from the predicted latent state. When part of the scene is masked, the model has to rely on motion priors it learned during pretraining to "see through" the obstruction.

What I Built

  • End-to-end pipeline that takes an 8-frame video clip with a vertical occlusion mask and predicts the projectile's position at each timestep.
  • V-JEPA 2.1 backbone running in predict mode over masked input patches, producing latent representations of the occluded region.
  • Lightweight CNN decoder mapping the predicted latents back into a spatial heatmap, then to (x, y) coordinates.
  • Custom dataset of projectile-motion sequences with controlled mask positions for evaluation.
  • Visualization tooling: ground-truth markers (green +), predicted markers (red +), per-frame attention heatmaps, and final overlay onto the original video.

Why It Matters

Most object trackers fall apart when the target leaves the frame or gets occluded. But in real-world tracking — including defense applications like munitions tracking, target re-acquisition, and sensor fusion — the object you care about doesn't stop existing just because you stopped seeing it. This project is a small step toward trackers that can carry physical priors through visual gaps.
